## **SECTION 11: DESIGN & ANIMATION GUIDELINES FOR CLAUDE CODE**

### **Color Palette**
```
Primary: #1a1a2e (Deep Navy) - Professional, trustworthy
Secondary: #16213e (Darker Navy) - Depth and contrast
Accent: #0f3460 (Rich Blue) - Highlights and CTAs
Success: #00d4ff (Cyan) - Positive actions, achievements
Text: #e0e0e0 (Light Gray) - Primary text
Muted: #a0a0a0 (Medium Gray) - Secondary text
Background: #0f0f1e (Almost Black) - Clean, modern
```

### **Typography**
```
Headings: Inter, Poppins, or similar modern sans-serif
- H1: 48px, bold, letter-spacing: -0.5px
- H2: 36px, bold
- H3: 24px, semi-bold
- H4: 18px, semi-bold

Body: Inter, Roboto, or similar
- Regular: 16px, line-height: 1.6
- Small: 14px, line-height: 1.5

Code/Technical: Fira Code, Courier New
```

### **Motion & Animation**
```
1. Scroll Animations
   - Fade-in on scroll (opacity 0 → 1)
   - Slide-in from sides (translateX)
   - Scale-up on scroll (transform: scale)
   - Staggered animations for lists/grids

2. Hover Effects
   - Card elevation (box-shadow increase)
   - Color transitions (smooth 0.3s)
   - Icon rotation or scale
   - Underline animations for links

3. Page Transitions
   - Smooth fade between sections
   - Parallax scrolling on hero
   - Floating particles or background animation

4. Interactive Elements
   - Button hover: Slight scale + color change
   - Links: Underline animation on hover
   - Forms: Focus states with color change
   - Expandable cards: Smooth height transition

5. Loading States
   - Skeleton screens for content
   - Animated loading spinner
   - Progress indicators

Timing: 0.3s for quick interactions, 0.6s for page transitions
Easing: ease-in-out for smooth, natural motion
```

### **Responsive Design**
```
Breakpoints:
- Mobile: 320px – 640px
- Tablet: 641px – 1024px
- Desktop: 1025px+

Mobile-First Approach:
- Stack all sections vertically on mobile
- Single column for cards/grids
- Touch-friendly buttons (44px+ tap targets)
- Simplified navigation (hamburger menu)

Tablet:
- 2-column grids where appropriate
- Larger spacing and padding
- Full navigation visible

Desktop:
- 3-column grids for skills/projects
- Side-by-side layouts for content
- Full-width hero section
- Sticky navigation
```

### **Accessibility**
```
- Semantic HTML (header, nav, main, section, footer)
- ARIA labels for interactive elements
- Color contrast ratio ≥ 4.5:1 for text
- Keyboard navigation support
- Focus indicators visible
- Alt text for images
- Form labels associated with inputs
```
